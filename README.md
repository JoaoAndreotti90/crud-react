# Projeto CRUD com React e Node.js

Este é um projeto de um CRUD (Create, Read, Update, Delete) de usuários, com o front-end construído em React e o back-end (API) em Node.js.

## Como Rodar o Projeto

O projeto é dividido em `frontend` e `backend`. Você precisará de dois terminais para rodar a aplicação completa.

### Rodando o Back-end (API)
1. Navegue para a pasta do back-end:
   `cd backend`
2. Instale as dependências:
   `npm install`
3. Inicie o servidor (geralmente com `json-server` ou `node`):
   `npm start`

### Rodando o Front-end
1. Em **outro terminal**, navegue para a pasta do front-end:
   `cd frontend`
2. Instale as dependências:
   `npm install`
3. Inicie a aplicação React:
   `npm start`

A aplicação estará disponível em `http://localhost:3000`.
